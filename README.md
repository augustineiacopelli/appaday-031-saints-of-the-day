# Saints of the Day — AppADay #031

A daily companion to the Roman Martyrology. Browse the principal saint for any day of the year — their biography, a signature quote, and a monthly collect prayer — with full 366-day coverage.

**Live:** <https://augustineiacopelli.github.io/appaday-031-saints-of-the-day/>  
**Portfolio:** <https://augustineiacopelli.github.io/appaday/>

-----

## What It Does

Displays the primary saint or feast for any day of the liturgical year, drawn from the General Roman Calendar and the Roman Martyrology. Each entry includes a biographical summary, a sourced quote, the liturgical season, and a month-specific collect prayer. A date picker and prev/next navigation allow browsing any date.

## Features

- Full 366-day coverage (January 1 – December 31, including February 29)
- One primary saint per day — the saint the date is known for
- Biographical summary, signature quote with attribution, era and region pills
- Liturgical season tag (Advent, Christmas Time, Lent, Easter Time, Ordinary Time)
- Month-specific collect prayer
- Jump to Date picker — always defaults to today on load
- Prev/Next day navigation (±60 days)
- Mobile-first, single-file, no dependencies

## Tech

Vanilla HTML, CSS, JavaScript — no frameworks, no build step, no API calls. All saint data is hardcoded from verified Catholic sources (General Roman Calendar, Roman Martyrology). Single file deployable directly to GitHub Pages.

## Sources

All entries drawn from or verified against the General Roman Calendar, the Roman Martyrology, the Roman Breviary, and standard hagiographic references. No entries were invented; biographical details not historically certain were omitted.
